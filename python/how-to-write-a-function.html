How to write a Function in Python

By Justin S A | Last updated: November 12, 2022

Python functions are written witn two keywords, def and return. the key word def identifies starting of a method.return keyword returns a field if it's applicable, there is no expilcit return type in python. use the correct indent while writing a function.

functions.py 
                            
                                def add(num1,num2):
                                    num3 = num1 + num2
                                    return num3
                                
                                def average(num1,num2):
                                    # add function call from another method
                                    num3 = add(num1,num2)
                                    avg = num3 / 2
                                    return avg
                                
                                #function add called from global scope.
                                print(add(30,40)) #prints 70
                                print(average(30 40)) #prints 35

For executing a function you can call directly from main or you can call the function from another function.
