How to create modules in Python

By Justin S A | Last updated: November 12, 2022

A module is a simple python programe, that is kept inside a package.for including a simple programme in to your current code use the import statement as below. 

                            import 
                            example:
                                import adder    #includes 'adder.py' in to your code.

A package is a folder that contains one or more Python modules. It must also contain a special module called __init__.py. Here is an example of a package so that you can create a package without any hassel. example: mypackage | | |-> __init__.py |-> module1.py |-> module2.py | |--->subpackage | ->module3.py

in the above example mypackage is a folder which contains __init__.py module1.py,module2.py programmes(modules).subpackage is a subfolder and it contains another module(programme) module3.py.

When including these modules in to other code blocks we need to declare the following import statements. 

                            import                  # Import all of ’s namespace into
                                                            # the name . Import module names
                                                            # can be accessed from the calling module
                                                            # with ..
                            import  as  # Import all of ’s namespace into
                                                            # the name . Import module
                                                            # names can be accessed from the calling
                                                            # module with ..
                            from  import , , ...
                                                            # Import only the names , ,
                                                            # etc, from . The names are added
                                                            # to the calling modules’s local namespace
                                                            # and can be accessed directly.

classes.py

In the above example the Desktop.py is kept under the folder hardware, so we are importing computer module using 'from hardware import computer' import statement.
