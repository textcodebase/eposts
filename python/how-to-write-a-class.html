How to write classes and intefaces in Python

By Justin S A | Last updated: November 12, 2022

interfaces,abstract classes and classes are integral part of any object oriented programming(OOP) language.python is no Exception to that.

Following example shows how we can write interfaces,abstract classes and derived classes.here in this example,Divice is an interface,computer and Desktop are its implementing classes.discription method is the abstract method which is overriden in all the implementation classes.

Here in the main section, we can see the polymorpic behavier of the device's instance.

classes.py 
                            
                                #interface
                                class Device:
                                    def description(self):
                                        pass
                                
                                #derived class from device
                                class Computer(Device):
    
                                    def __init__(self,processor,ram):
                                        self.processor = processor
                                        self.ram = ram

                                    def description(self):
                                        print(f"This computer has {self.processor} Processor and {self.ram} of RAM")
                                
                                #derived class from computer class
                                class Desktop(Computer):
                                    def __init__(self, processor, ram,keybord,mouse):
                                        super().__init__(processor, ram)
                                        self.keybord = keybord
                                        self.mouse = mouse
                                        
                                    def description(self):
                                        print(f"This Desktop has {self.processor},{self.ram} {self.keybord} and {self.mouse}")
            

                                computer = Computer("Intel 12gen eon","32 GB")
                                computer.description()
                                #print(computer.processor)
                                
                                device = Desktop("Intel 12gen eon","32 GB","TVS keybord","Logitek mouse");
                                device.description()
