PDF Content Extractor using PDF Box Library

By Justin S A | Last updated: November 06, 2022

The below sample code extracts content from a pdf file. for this we need to use pdf-box dependence to the maven pom File 

                        
                            org.apache.pdfbox
                            pdfbox
                            2.0.27
PDFExtractor.java 
        
                            
                                package com.text.eposts.io;

                                import java.io.File;

                                import org.apache.pdfbox.pdmodel.PDDocument;
                                import org.apache.pdfbox.text.PDFTextStripper;

                                public class PDFExtractor {
                                    
                                    private PDDocument document = null;
                                    
                                    public PDFExtractor(String file) throws Exception {
                                        super();
                                        this.document = PDDocument.load(new File(file));
                                    }
                                    public String getPageContent(int pageNo)throws Exception {
                                        PDFTextStripper stripper = new PDFTextStripper();
                                        stripper.setStartPage(pageNo);
                                        stripper.setEndPage(pageNo);		
                                        String text = stripper.getText(document);
                                        return text;
                                    }
                                    public static void main(String[] args) throws Exception {
                                        PDFExtractor extractor = new PDFExtractor("sample.pdf");
                                        System.out.println(extractor.getPageContent(4));
                                    }
                                    
                                }
